## The decision

**Recognition is the fundamental need; Memory is a mechanism that expresses it.**
"The Club remembers me" matters because it proves the deeper thing: *I mattered.*

## Links

- **Import Packages:** IP-002 A2 (Recognition), CE-1 (the compression: Recognition
  moved beneath Memory), CR-1. IP-001 AT-9 (the two memories) is reclassified from
  a first principle to a mechanism.

## Alternatives considered

- **Memory as a first principle** — the earlier framing, in which the
  Historic/Living memory model was treated as bedrock.
- **Recognition as merely one expression of memory** (the reverse ordering).

## Why this won

Memory is **evidence**; recognition is the human **need** that evidence points to.
Reciprocal memory — "the Club remembers me" — is powerful precisely because it is
proof of *mattering*, which is a fundamental, well-studied human need (recognition
/ "mattering"). One-directional memory ("I remember the Club") is nostalgia and
consumption; it can be felt about a film. Bi-directional memory is *relationship*.
So memory sits **beneath** recognition, not the other way round — and the deeper
of the two is the need, not the evidence.

## What it rejects or supersedes

- **Demotes** "Memory as a first principle" to a *mechanism* that expresses
  Recognition (and Meaning — see CD-004).

## Consequences

- Historic Memory and Living Memory become mechanisms serving Recognition and
  Meaning, not standalone principles.
- The living world's "the world remembers and discusses you" (Victor, Emma, Nora)
  is recognition machinery, not decoration.
- Promotes the **Human Test** toward a load-bearer: it is the tether that keeps
  internal recognition channelling real human regard.
